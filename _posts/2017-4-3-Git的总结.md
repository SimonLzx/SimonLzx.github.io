## 创建Git的版本库
<pre class="brush: shell">
git init
</pre>
## 创建一个新的分支
<pre class="brush: shell">
git branch dev
</pre>
## 切换到另一个分支
<pre class="brush: shell">
git checkout master
</pre>
## 查看当前仓库的状态
<pre class="brush: shell">
git status
</pre>
## 查看具体修改了什么内容
<pre class="brush: shell">
git diff #diff顾名思义九十difference，显示的格式正式Unix通用的diff格式
</pre>
## 查看git的历史记录
<pre class="brush: shell">
git log #命令显示从最近到最远的提交日志，我们可以看到3次提交，最近的一次是`append GPL`，上一次是`add distributed`，最早的一次是`wrote a readme file`s
</pre>
